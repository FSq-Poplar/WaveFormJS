WaveForm.js is a customizable audio waveform visualization, built on top of the Web Audio API.

Users are able to navigate through the music/audio by clicking on the visualizer; other controls can be designed using the WaveForm.js api.

Unlike existing audio visualizers, WaveForm.js does not use the HTML5 Canvas and instead creates a visualization using CSS classes created and passed by the user. This allows for a far greater level of customization, at the cost of greater performance overhead.

Quick Start

Import the library into your web page:

<script defer type="text/javascript" src='./Waveform.js'></script>

Create a container where the waveform is to appear:

<div id="visualizerContainer"></div>

In JavaScript, create a new SingleWaveForm object and select its container, audio source, bar count and class names:

// Create new WaveForm object
var visualization = new SingleWaveForm(
    containerID = "visualizerContainer", 
    audio = "./path_to_audio/audio.mp3", 
    barCount = 64, 
    barClassNotPlayed = "firstClassName",
    barClassIsPlayed = "secondClassName",
)

From there, controls can be constructed using the WaveForm object's API:

// Add play button
document.getElementById("playButton").addEventListener('click', function() {
    visualization.playAudio()
})

// Add change color button
document.getElementById("changeColorButton").addEventListener('click', function() {
    visualization.changeNotPlayedBarClass("thirdClassName")
    visualization.changeIsPlayedBarClass("fourthClassName")
})

Browser Support

WaveForm.js should run on all modern browsers supporting Web Audio.
