Constructor

Currently, the three visualization designs SingleWaveForm, DoubleWaveForm and CircleWaveForm all share the same constructor and parameters, making them easily interchangeable: 

// Creating a new WaveForm.js visualization
var myWaveForm = new WaveForm(
    containerID: string,
    audio: string,
    barCount: int,
    barClass: string
)

where

  • containerID (string) is the html ID of the container the visualizer will be built in
  • audio (string) is a string representing the path to, or CORS header to, the audio file
  • barCount (int) is some power of 2 representing the number of frequency bands represented
  • barClass (string) is the CSS class name applied to sections of audio that have not been passed

Note that while containerID is not required to be empty, the WaveForm object will be built as if the container is empty and any inner HTML (such as controls) should define a z-index to appear on top, or be interactable/clickable.

Additionally, any CSS applied to the container element will not be affected; this means sizes, background images etc. are fully in control of the user. Such applications can be seen on the examples page.

Options

WaveFormJS will parse the following options:

  • barClassPlayed (string) is the CSS class name applied to sections of audio that have been passed
  • clickable (bool) determines whether clicking on the visualizer's bars navigates audio
// Creating a visualization with optional arguments
var myWaveForm = new WaveForm(
    containerID: string,
    audio: string,
    barCount: int,
    barClass: string
    options: {
        option: value
    }
)

API

After creating a WaveForm.js visualizer (with var myWaveForm = new WaveForm(...)) the following methods can be called to create controls or modify the visualization and sound:

  • playAudio() - Begins or resumes audio playback.
  • stopAudio() - Pauses audio playback.
  • navigate(seconds) - Navigates to time seconds in the audio, if possible.
  • navigateRatio(ratio) - For 0 ≤ ratio ≤ 1, sets playhead to ratio times the total duration.
  • navForward(seconds) - Moves playhead seconds, or stops playback if past the end.
  • navBackward(seconds) - Moves playhead seconds backwards, or the beginning if the destination time is before 0.
  • changeVolume(level) - For 0 ≤ level ≤ 1, sets the volume to the provided level.
  • incrementVolume(increase) - Increases the volume by increase, to a maximum of 1.
  • decrementVolume(decrease) - Decreases the volume by decrease, to a minimum of 0.
  • toggleMute() - Mutes volume. Note this does not stop playback, and the playhead will continue to move!
  • changeAudioSource(newSource) - Sets audio source to the file at or CORS header newSource. Resets playback.
  • changeBarClass(newClass) - Changes the class name applied to not played audio to string newClass.
  • changeBarClassPlayed(newClass) - Changes the class name applied to played audio to string newClass.
